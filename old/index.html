Documentation for Adaptive LibPFASST

This page provides the documentation of Jan Fecht's interdisciplinary project at the Technical University of Munich 2022-2023.

Multiple software components were developed or modified in the course of this project. Please use the navbar to find the respective documentation. The following software components are part of the project:

  • Open MPI: Patrick Huber's adaptive Open MPI fork was extended with a Fortran interface.
  • libmpidynres: A Fortran interface was added to libmpidynres.
  • LibPFASST: The main component of this project
  • tmpi.py: A terminal program for debugging MPI applications with dynamic MPI support.
